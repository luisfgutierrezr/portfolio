## Overview

This project involved the development of a comprehensive web application designed to streamline the management of property rentals. The application was built with a focus on user-centric design and robust functionality, ensuring a seamless experience for both property managers and tenants.

### Key Responsibilities

- **User Story Approach:** Employed user stories to define and prioritize features, ensuring that the final product met the specific needs of its users.
- **Application Design:** Designed the user interface and user experience using Figma, creating intuitive and visually appealing layouts.
- **Backend Development:** Connected the application to a MySQL database using Spring Boot, developing a resilient and scalable backend architecture.
- **Frontend Development:** Implemented the user interface with Angular, building a responsive and interactive frontend that delivers a smooth user experience.
- **Testing:** Conducted thorough unit, end-to-end (E2E), and automated testing to ensure the application's reliability and performance.

### Technical Stack

- **Frontend:** Angular, HTML, CSS, JavaScript, TypeScript
- **Backend:** Spring Boot
- **Database:** MySQL
- **Design and Prototyping:** Figma
- **API Testing:** Postman
- **Architecture:** MVC (Model-View-Controller)

### Key Features

- **User Management:** Complete user registration and authentication system
- **Property Listings:** Comprehensive property management with search and filter capabilities
- **Booking System:** Integrated booking and reservation management
- **Payment Integration:** Secure payment processing for rentals
- **Admin Dashboard:** Full administrative interface for property managers
- **Responsive Design:** Mobile-first approach ensuring accessibility across all devices

### Development Process

The project followed an agile methodology with user stories to ensure all requirements were met. The development process included:

1. **Requirements Analysis:** Gathering user stories and defining project scope
2. **UI/UX Design:** Creating wireframes and prototypes using Figma
3. **Database Design:** Designing and implementing the MySQL database schema
4. **Backend Development:** Building RESTful APIs with Spring Boot
5. **Frontend Development:** Creating responsive interfaces with Angular
6. **Testing:** Comprehensive testing including unit, integration, and E2E tests
7. **Deployment:** Production deployment and monitoring setup
